Git

  • git status - get current working branch and all document changed files
  • git add - add files to the active branch
    • -A - will add all files in the folder to the branch
    • <filename> - to add specific files
    • <directory>/ - will add the directory and all files in the directory
    • <directory>/<filename> - will include the directory and just the listed file
  • git commit -m "<message>" - updates the current working branch with all files marked for change give a discriptive message
  • git push origin <branch-name> - tells GitHub that you want to merge the current branch to its parent
  • git checkout -b <branch-name> - makes a new branch with the specified name
